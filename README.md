# EPUB Audio Reader

Leitor de EPUB com TTS (Text-to-Speech) offline para Android.

## Funcionalidades
- Leitura de EPUB com biblioteca e capítulos
- TTS 100%% offline com Sherpa-ONNX
- Voz em português do Brasil (pt-BR)
- Leitura por parágrafo com segmentação inteligente
- Prefetch de segmentos para playback contínuo
